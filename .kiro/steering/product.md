# Product Overview

**findhub** is a full-stack TypeScript application built with modern web technologies. It's a monorepo project that separates frontend (Next.js) and backend (Hono) concerns while sharing common packages for authentication and database access.

## Key Characteristics

- Full-stack TypeScript application with type safety throughout
- Progressive Web App (PWA) support for offline-first capabilities
- Authentication-enabled platform using Better-Auth
- PostgreSQL-backed data persistence
- Monorepo architecture for code sharing and modularity
